# PRD — DHL IT Onboarding Portal

## 목적

DHL을 처음 사용하는 고객이 자신의 비즈니스 상황에 맞는 DHL 온라인 솔루션을 쉽게 찾을 수 있도록 AI가 맞춤 추천을 제공하는 웹 서비스.

## 추천 대상 솔루션

| 솔루션 | 추천 대상 |
|--------|-----------|
| MyDHL+ | 시스템 연동 없이 빠르게 DHL 발송을 시작하고 싶은 고객 |
| DEC | 온라인 쇼핑몰, 이커머스 중심의 대량 발송 고객 |
| MyDHL API | 자체 IT 인프라를 보유한 기업 (ERP/WMS 연동 자동화) |

## 페이지 구성

### 1. 홈 페이지 (입력 폼)

사용자에게 6가지 질문:
1. 업종 (예: 의류 판매상, 반도체 제조사 등)
2. 주요 발송물 (예: 의류, 반도체 칩)
3. 월 발송 건수 (숫자 입력)
4. 출발 국가
5. 도착 국가
6. IT 시스템/개발팀 보유 여부 (있음/없음/모르겠음)

각 질문에 예시 힌트 문구 포함.
이전 결과 조회 기능: 6자리 숫자 코드 입력.

### 2. 결과 페이지 (`/result/[key]`)

- 추천 솔루션 1개 (AI 근거 포함)
- 3개 솔루션 각각의 고정 설명 (추천 솔루션 강조)
- 루트별 통관 주의사항 2가지 (AI 생성, 출발국가→도착국가 구간 특화, 통관·세금·발송 제한 관련 critical 정보)
- 통관 용어 안내 섹션 (HS코드, 인코텀즈 등 기본 설명)
- 6자리 조회 코드 표시 + 복사 버튼
- 이메일 발송 버튼

## 비기능 요구사항

- 별도 로그인 없음
- 6자리 코드로 이전 결과 재조회 가능 (영구 보존)
- DHL 브랜드 디자인 (#FFCC00, #D40511)
- 한국어 UI
